# GuiLocker
GuiLocker is DisLocker wrapper with GUI

## Demo
[![Watch the video](https://img.youtube.com/vi/ltqnQjCR27A/maxresdefault.jpg)](https://youtu.be/ltqnQjCR27A)

## Usage
1. install python-tk, dislocker
2. launch main.py
3. inject usb
